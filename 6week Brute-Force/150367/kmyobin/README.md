## 풀이
숫자가 주어졌을 때, 그 숫자를 더미 노드를 이용한 포화 이진트리로 변환할 수 있는지 묻는 문제였다.

1. 숫자를 이진수로 변환한다.
2. `포화 이진트리`는 N줄일 때 `(2^N)-1`이기 때문에 만든 이진수 앞부분에 부족한 만큼 0을 채워넣는다.
3. `이진 탐색`을 통해 부모 노드가 0일 때 자식 노드가 1인 경우를 찾아낸다.

이 과정을 반복하여 풀었다.


## 마무리
로직은 구성하였으나 이진 탐색의 범위를 틀려서 고생했다.
