## ❗풀이
- 나머지 연산을 해서 쭉 반복되는 패턴들을 정답과 대조하는 방식이 핵심인데, 단순하게 if문으로 일치하면 각 학생 별로 맞춘 정답 수를 저장하고, 가장 많이 맞춘 사람의 정답 수를 구하여 이 max값과 일치하는 학생들의 index를 return 하도록 했다.

## 마무리
더 쉬운 방법이 있을 것 같은데 if문으로 단순 구현해서 아쉽다.
