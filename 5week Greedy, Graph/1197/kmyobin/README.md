## Info
<a href="https://www.acmicpc.net/problem/1197" rel="nofollow">1197 최소 스패닝 트리</a>


## 풀이
`최소 스패닝 트리(MST)`문제이다. `최소 스패닝 트리`란 `그래프 내의 모든 정점을 거치고 cycle을 형성하지 않으면서 가중치의 합이 최소인 트리`이다. 

대표적으로는 `Prim` 알고리즘과 `Kruskal` 알고리즘이 있는데, 학교에서 알고리즘 설계를 수강했던 당시 `Prim`의 동작 방식을 더 깊히 공부했기 때문에 `Prim`을 선택하였다.

한 번 방문했던 정점은 다시 방문하면 안되기 때문에 `bool visited[10001]`을 선언하여 설정해주었다.

`Edge` 구조체를 만들어서 어디로 향하는지(`to`), 가중치는 얼마인지(`weight`)를 담았다. 

`우선순위 큐`로 구현하였는데, 이는 기본으로 내림차순(`최대힙`)으로 정렬하기 때문에 작은 가중치를 우선시 하는 이 문제에서는 `최소힙`으로 변경할 필요가 있었다. 따라서 `연산자 오버로딩`을 이용하여 오름차순으로 변경하였다. 따라서 `pq`를 `pop`할 때마다 제일 작은 가중치를 가진 `Edge`가 나올 수 있는 것이다.

## 마무리
작년에 배웠던 `MST`가 나와 반가웠지만 직접 알고리즘을 짜려니 힘들었다. 블로그에 있는 코드들을 참고하여 작성하였다.
