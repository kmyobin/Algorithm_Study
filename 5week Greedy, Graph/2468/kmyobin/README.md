## Info
<a href="https://www.acmicpc.net/problem/2468" rel="nofollow">2468 안전 영역</a>


## 풀이
`BFS`로 문제를 풀었다.

3중 for문을 작성했기에 시간 초과가 나지 않을까 걱정했지만 발생하지 않았다.

`BFS` 함수에 `cnt` 변수를 넣어 안전 영역의 개수를 세고, 최고 높이가 `k`일 때를 모두 계산해서 최댓값을 `maximum`에 넣어줬다.

`k`는 1부터 시작하면 안되고 0부터 시작해야 한다. 노트에 `아무 지역도 물에 잠기지 않을 수도 있다.`라는 말이 있기 때문이다.


## 마무리
`BFS` 문제를 다양하게 풀 수 있어서 좋았다.
